## 🎯 Quick Reference

### Installation
```bash
1. Go to chrome://extensions/
2. Enable "Developer Mode"
3. Click "Load unpacked"
4. Select this folder
5. Done! ✨
```

### Three Ways to Clean

| Method | Shortcut | Speed | Best For |
|--------|----------|-------|----------|
| **Popup UI** | Click icon | Medium | Custom URLs |
| **Quick Button** | "Clean Current Site" | Fast | Current website |
| **Keyboard** | Ctrl+Shift+L | Fastest | Quick access |
| **Context Menu** | Right-click | Fast | Easy access |

### Input Examples
```
google.com                    → Clean entire domain + subdomains
github.com/settings           → Clean specific path only
https://twitter.com/home      → Clean this URL
example.com                   → Clean with path: /
```

### What Gets Cleaned
```
✓ History        - All visited pages
✓ Cookies        - Login sessions, tracking
✓ Cache          - Images, scripts, storage
```

## ❓ Common Questions

### Q: Is my data safe?
**A:** Yes! All operations happen locally. No data is sent anywhere.
→ Read: [README.md - Privacy & Security](README.md#-privacy--security)

### Q: How do I install it?
**A:** 4 simple steps in chrome://extensions/
→ Read: [QUICKSTART.md](QUICKSTART.md)

### Q: How does domain matching work?
**A:** Matches domain + all subdomains, with optional path prefix.
→ Read: [ARCHITECTURE.md - Domain Matching](ARCHITECTURE.md#️-domain-matching)

### Q: What if something doesn't work?
**A:** Check the troubleshooting section.
→ Read: [INSTALL.md - Troubleshooting](INSTALL.md#-troubleshooting)

### Q: Can I use a custom keyboard shortcut?
**A:** Yes! Go to chrome://extensions/shortcuts
→ Read: [README.md - Keyboard Shortcuts](README.md#⌨️-keyboard-shortcuts)

### Q: Does it work on other browsers?
**A:** Yes! Any Chromium-based browser (Chrome, Edge, Brave, etc.)
→ Read: [README.md - Installation](README.md#-installation)

### Q: What gets deleted?
**A:** History, cookies, and cache for the specified website.
→ Read: [QUICKSTART.md - What Gets Deleted](QUICKSTART.md#what-gets-deleted)

### Q: Is it free?
**A:** Yes! Completely free, no ads, no tracking.
→ Read: [README.md - Privacy & Security](README.md#-privacy--security)
