# 🧹 All-in-One Cleaner - Browser Extension

> **A lightweight, privacy-focused browser extension that cleans history, cache, and cookies for specific websites.**

![Status](https://img.shields.io/badge/status-complete-brightgreen)
![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Browser Support](https://img.shields.io/badge/browsers-Chrome%20%7C%20Edge%20%7C%20Brave-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🎯 What This Does

Clean **history**, **cookies**, and **cache** for any website in seconds!

### Example:
```
You want to clean amazon.com
├─ Enter: "amazon.com"
├─ Click: "Clean Now"
└─ Result: ✅ All amazon data deleted
    - 156 history entries
    - 23 cookies
    - Cache cleared
```

## ✨ Features

| Feature | Details |
|---------|---------|
| 🧹 **History Cleaner** | Remove all visits to a domain |

## 🚀 Quick Start (30 seconds)

### Step 1: Navigate
Go to `chrome://extensions/`

### Step 2: Enable Developer Mode
Click the toggle in the **top-right corner**

### Step 3: Load Extension
Click **"Load unpacked"** → Select this folder

### Step 4: Done!
You now have the extension! 🎉

---

## 📖 How to Use

### Method 1: Popup UI
1. Click extension icon
2. Enter website: `example.com`
3. Click "Clean Now"

### Method 2: Quick Button
1. Click extension icon
2. Click "Clean Current Site"

### Method 3: Keyboard Shortcut
- **Windows/Linux**: `Ctrl+Shift+L`
- **Mac**: `Cmd+Shift+L`

### Method 4: Context Menu
1. Right-click on any webpage
2. Select "Clean This Site"

## 🧠 How It Works

### History Matching
```
Input: example.com/blog/
Matches:
  ✓ example.com/blog/
  ✓ example.com/blog/post-1
  ✓ www.example.com/blog/
  ✗ example.com/news/
```

### Domain Matching
```
Input: example.com
Matches:
  ✓ example.com
  ✓ www.example.com
  ✓ api.example.com
  ✓ any.subdomain.example.com
```

## 🔍 Technical Details

### Permissions
```json
{
  "history",        // Read/delete history
  "cookies",        // Read/delete cookies
  "browsingData",   // Clear cache
  "contextMenus"    // Right-click menu
}
```

### APIs Used
- `chrome.history` - History management
- `chrome.cookies` - Cookie management
- `chrome.browsingData` - Cache clearing
- `chrome.contextMenus` - Context menu
- `chrome.commands` - Keyboard shortcuts
- `chrome.runtime` - Message passing
- `chrome.storage` - Preference saving
- `chrome.tabs` - Tab information

### No Dependencies
✅ Pure JavaScript
✅ No libraries needed
✅ No external packages
✅ Minimal code
