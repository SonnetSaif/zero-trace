# ⚡ Quick Start Guide

## 30-Second Setup

### 1️⃣ Open Extensions Page
- **Chrome**: `chrome://extensions/`
- **Edge**: `edge://extensions/`
- **Brave**: `brave://extensions/`

### 2️⃣ Enable Developer Mode
- Click the toggle switch in the **top-right corner**

### 3️⃣ Load Extension
- Click **"Load unpacked"**
- Select this folder
- Click **"Open"** (or similar)

### 4️⃣ Done! 🎉
- You'll see "All-in-One Cleaner" in your extensions list
- Icon appears in your toolbar

---

## 3 Ways to Use

### Way 1: Popup (Easy for Custom URLs)
```
1. Click extension icon
2. Enter: example.com
3. Click "Clean Now"
✓ Done!
```

### Way 2: Quick Button (Easiest)
```
1. Click extension icon
2. Click "Clean Current Site"
✓ Instantly cleans the website you're on!
```

### Way 3: Keyboard Shortcut (Fastest)
```
Press: Ctrl+Shift+L  (Windows/Linux)
Press: Cmd+Shift+L   (Mac)
✓ Done! (cleans current website)
```

## Input Examples

### Clean Entire Domain
```
Input: google.com
Cleans: 
  • google.com
  • www.google.com
  • mail.google.com
  • any.subdomain.google.com
```

### Clean Specific Path
```
Input: github.com/settings
Cleans:
  • github.com/settings
  • github.com/settings/profile
  • github.com/settings/security
  
Does NOT clean:
  • github.com/notifications
```

### Clean HTTPS URL
```
Input: https://twitter.com/home
Same as: twitter.com/home
```

## Need Help?

### Popup doesn't appear?
- Make sure extension is enabled in `chrome://extensions/`
- Try clicking the extension icon again

### Shortcut doesn't work?
- Go to `chrome://extensions/shortcuts`
- Verify shortcut is enabled
- Choose a different key if it conflicts

### Nothing cleaned?
- Check you entered the correct domain
- Verify checkboxes are checked
- Try "Inspect popup" to see errors (right-click icon)

### Want to customize shortcuts?
1. Go to `chrome://extensions/shortcuts`
2. Find "All-in-One Cleaner"
3. Click on the shortcut field
4. Enter your preferred keys
5. Click Save
